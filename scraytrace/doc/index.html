Raytracing Software for the Simulation of the Solar Corona: User's Guide

Abstract

This is the documentation for the Solar Corona Ray-Tracing software developed in the solar physics team of the Naval Reasearch Lab. This document provide a quick overview of the raytracing software concepts, an installation guide and a small tutorial. There is also a section that gives more details about the implementation for the user who would like to contribute to the project. Finally a quick description of the software programs is given. Nevertheless, users might have to refer to the source code documentation for more details about the program parameters.
